# 越南峴港之旅 2026 (Vietnam Da Nang Travel App)

這是一個專為 2026 年越南峴港旅遊所打造的專屬 Web 應用程式 (旅遊指南 & 行程表)。本專案旨在提供一個直觀、美觀且實用的旅遊資訊中心，讓旅伴在旅途中能隨時查看行程、實用工具以及旅遊須知。

## 🌟 主要功能 (Features)

* **📅 互動式行程表 (Itinerary)**
  * 每日行程總覽與詳細時間軸。
  * 景點介紹與交通方式（包含去程與回程資訊）。
  * 餐廳與住宿（含 Google 評論亮點與早餐介紹）資訊整合。
* **🧳 旅遊指南 (Travel Guide)**
  * 出發前必看：行前準備與分類旅遊提示 (簽證、網卡、天氣、穿著、文化禁忌等)。
  * 符合 Anthropic 品牌視覺設計規範的質感情境排版。
* **🛠️ 實用工具 (Tools)**
  * **越南盾匯率換算器**：動態即時計算，支援 LocalStorage 保存狀態。
  * **越南語常用手冊 (Phrasebook)**：收錄旅遊常用語句、拼音標註，並內建**發音音檔播放**功能。

## 💻 技術棧 (Tech Stack)

本專案使用現代化前端技術打造，兼顧高效能開發體驗與流暢的使用者互動：

* **框架/核心**：[React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
* **語言**：TypeScript
* **路由**：[React Router DOM](https://reactrouter.com/)
* **樣式與排版**：[Tailwind CSS (v4)](https://tailwindcss.com/) + `clsx` & `tailwind-merge`
* **動畫效果**：[Framer Motion](https://motion.dev/)
* **圖示庫**：[Lucide React](https://lucide.dev/)
* **部署**：GitHub Pages (`gh-pages`)

## 🎨 設計系統

外觀遵循乾淨、具現代感且能驚艷使用者的 UI 設計原則。設計上特別強調：

* 自定義的排版與字體 (Poppins / Lora 等) 以提升閱讀體驗。
* 豐富的微動畫 (Micro-animations) 與流暢的畫面過渡。
* 增強對比度使重要資訊（時間、交通）一目瞭然。

## 📝 備註

這是一個為家人/朋友一同前往峴港旅遊量身打造的作品，專案內充滿對行程的期待與規劃細節。Enjoy the trip! 🇻🇳
